## Quick Start

```bash
# 1. Stand up infrastructure
cd infra/
terraform init
terraform plan -var-file=terraform.tfvars
terraform apply -var-file=terraform.tfvars

# 2. Deploy application stack
kubectl apply -f app/namespace.yaml
kubectl apply -f app/database/
kubectl apply -f app/redis/
kubectl apply -f app/frontend/
kubectl apply -f app/hydrate/

# 3. Run validation tests
cd test/
pip install -r requirements.txt
pytest -v

# 4. Set up ops layer
cd ops/
# Follow ops/README.md
```

## Design Principles

1. **Deterministic Operations** вҖ” Log querying and metric retrieval use pre-defined, parameterized queries (not LLM-generated) for consistency and token efficiency
2. **Grounded Validation** вҖ” All agent outputs are validated against actual CloudWatch metrics and logs
3. **Repeatable Format** вҖ” Operations follow a structured, versioned format that can be replayed
4. **Token Efficiency** вҖ” Minimize LLM token usage by pre-processing and filtering data before agent consumption
